      Clone the repository and install dependencies using Composer.
      
      ```bash
      git clone https://github.com/TheCodeStudioxyz/teleman-telemarketing.git
      cd teleman-telemarketing
      composer install
      ```
      
      Learn more in our [installation guide](/installation).
    </Step>
    
    <Step title="Configure your environment">
      Set up your `.env` file with database credentials and Twilio API keys.
      
      ```bash
      cp .env.example .env
      php artisan key:generate
      ```
      
      See the complete [configuration guide](/configuration).
    </Step>
    
    <Step title="Create your first campaign">
      Access the dashboard and create a voice campaign with AI-powered calling.
      
      Navigate to **Dashboard** → **Campaigns** → **Create New** to get started.
      
      Follow our [campaign creation guide](/guides/creating-campaigns) for detailed instructions.
    </Step>
